# 📝 To-Do App

A simple and responsive To-Do application built using HTML, CSS, and JavaScript.

## Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Stores tasks using Local Storage
- 📱 Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ItzzPriyankagarg/to-do-App.git
   ```

2. Open `index.html` in your browser.
